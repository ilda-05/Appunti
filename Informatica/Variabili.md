
Le variabili si comportano come contenitori di informazioni.

Ci sono diversi tipi di variabili :
- **Int** : numeri interi
- **Float** : numeri reali
- **Char** : un singolo carattere
- **Array** : contiene più dati all'interno 
	![[array.excalidraw]]
- **String** : un array di caratteri
- **Boolean** : una variabile che ha 2 stati, che solitamente sono true o false
